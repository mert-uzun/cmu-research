**Property:** Every string in Set A contains **both** the substring `ab` **and** the substring `ba`.

**Verification:**

- Each Set A string has at least one place where an `a` is immediately followed by a `b` (an `ab`), *and* at least one place where a `b` is immediately followed by an `a` (a `ba`). This is equivalent to saying the string changes from a-block to b-block at least once *and* from b-block to a-block at least once — i.e., it consists of at least **three alternating runs** of a's and b's (e.g., `a…b…a…` or `b…a…b…`).

- Every Set B string fails this:
  - `aa`, `bb`, `aaaaa` — no letter change at all.
  - `ab`, `aab`, `aaaab`, `abbbbbbb` — contain `ab` but never `ba` (only two runs, a's then b's).
  - `ba`, `baaa`, `bbbbba` — contain `ba` but never `ab` (only two runs, b's then a's).

So Set B contains only strings with **at most two runs**, which never have both `ab` and `ba`.

**Succinct statement:** A string is in A iff it contains both `ab` and `ba` as substrings (equivalently, it has three or more alternating runs of identical letters).
