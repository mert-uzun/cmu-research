## Analysis

Let me look at the first character of each string together with its length.

**Set A:**
| string | starts with | length |
|--------|-----|--------|
| aa | a | 2 (even) |
| ab | a | 2 (even) |
| aaba | a | 4 (even) |
| abaabb | a | 6 (even) |
| abababbb | a | 8 (even) |
| b | b | 1 (odd) |
| bab | b | 3 (odd) |
| bbaba | b | 5 (odd) |
| baabbba | b | 7 (odd) |
| babbabbab | b | 9 (odd) |

Every string that **starts with `a`** has **even** length, and every string that **starts with `b`** has **odd** length.

**Checking Set B** — each one breaks this rule:
- `a`, `aab`, `ababa`, … start with `a` but have **odd** length ✗
- `bb`, `baab`, `bbabbb`, … start with `b` but have **even** length ✗

## Property

**A string is in set A if and only if it starts with `a` and has even length, OR starts with `b` and has odd length.**

Equivalently: *the string is non-empty, and it begins with `a` exactly when its length is even (begins with `b` exactly when its length is odd).*
